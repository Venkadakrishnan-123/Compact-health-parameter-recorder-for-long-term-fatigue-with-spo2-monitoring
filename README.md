# Compact-health-parameter-recorder-for-long-term-fatigue-with-spo2-monitoring
The Compact Health Parameter Recorder for Long-Term Fatigue with SpO₂ Monitoring is an IoT-based wearable healthcare system that continuously monitors heart rate, blood oxygen saturation (SpO₂), and body temperature using an ESP32 microcontroller with MAX30102 and DS18B20 sensors. The system processes and stores real-time health data, supports wireless monitoring through Wi-Fi, and performs fatigue detection by analyzing changes in vital signs. Its compact, low-power, and cost-effective design makes it suitable for remote patient monitoring, preventive healthcare, elderly care, and long-term health tracking.
